## 项目简介
AAIMS (Academic Affairs Information Management System) 是一个高效、结构化的教务信息管理方案，旨在提供严谨的学生学籍、课程及成绩管理功能。

## 已实现功能

### 1. 教务数据管理 (管理员端)
* **学生档案维护**：支持学号、姓名、性别、年龄、宿舍、联系方式等全维度信息录入。
* **动态课程管理**：
    * 管理课程编号、名称、课时及学分配置。
    * **高级成绩模型**：支持平时成绩、考试成绩与综合成绩的多权重计算。
    * **重修逻辑追踪**：完整支持重修学期记录、多次重修记录以及历史成绩溯源。

### 2. 选课与排课 (学生端)
* **自主选课系统**：学生可根据课程编号进行在线选课。
* **个人课表生成**：实时生成并展示学生的个性化周课表。

### 3. 成绩评价体系 (教师/学生端)
* **成绩批录**：教师端支持针对行政班级或选课名单批量录入成绩。
* **多维成绩查询**：学生可即时获取各科成绩详情及学分状态。

### 4. 统计与分析模块
* **班级维度统计**：按行政班级聚合选课分布及考试通过率。
* **课程维度统计**：按课程生成学生花名册，分析该科目的成绩分布。
* **学业预警**：自动计算已修总学分，并精准列出所有不及格课程。

### 5. 交互与架构
* **精准搜索**：支持通过唯一标识符快速检索学生完整的学术档案。
* **多端连接**：支持多端并发连接与数据交互。

## 构建指南

### 环境要求
* **编译器**: 支持 **C++23** 标准的编译器 (GCC 13+, Clang 15+, 或 MSVC 19.33+)。
* **构建系统**: [CMake](https://cmake.org/) (建议版本 3.20+ 以良好支持 C++23)。
* **依赖项**: [nlohmann/json](https://github.com/nlohmann/json) (自动获取)。

### 编译步骤
```bash
git clone [https://github.com/vlouboos/AAIMS.git](https://github.com/vlouboos/AAIMS.git)
cd AAIMS
cmake -B build -DCMAKE_CXX_STANDARD=23
cmake --build build
```
